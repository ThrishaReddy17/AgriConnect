AgriConnect - Agricultural E-commerce Platform

A comprehensive PHP-based e-commerce platform designed specifically for the agricultural sector, connecting farmers, organizations, and customers in a sustainable marketplace.

Key Features

  • Multi-role authentication (Farmers, Organizations, Customers, Admin)
  • Product management and marketplace
  • Stubble burning management system
  • Shopping cart and order processing
  • Admin dashboard with comprehensive controls
  • Multilingual support (English, Hindi, Telugu)
  • Responsive design for all devices
  • Secure payment processing
  • Notification system
  • Profile management for all user types

Technology Stack

Backend: PHP 7.4+, MySQL, PDO

Frontend: HTML5, CSS3, JavaScript

Features: Session management, File uploads, SMS integration (Twilio)

Security: Password hashing, SQL injection prevention, XSS protection
